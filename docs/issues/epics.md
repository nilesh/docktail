# Epics

## E1: Core Log Streaming
Stream logs from Docker Compose containers into a unified TUI view.

## E2: Navigation & Selection
Freeze logs and navigate, select, copy log lines with keyboard and mouse.

## E3: Search & Filtering
Text and regex search, log level filtering.

## E4: Container Management
Sidebar with container list, actions (start/stop/restart/pause), status display.

## E5: Integrated Shell
Docker exec shell panel below logs with tab bar and resize.

## E6: Polish & Release
Clipboard integration, tests, CI/CD, goreleaser, README polish.
